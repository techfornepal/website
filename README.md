# Tech For Nepal

This is the repository for the Tech For Nepal website. We are building a
community of tech enthusiasts, entrepreneurs, and innovators who are passionate
about empowering Nepal through technology, innovation, and community
collaboration.

## Repository Quickstart

```bash
git clone https://github.com/techfornepal/website.git
cd website
npm install  # installs dependencies + sets up code quality hooks
npm run dev  # start development server
```
